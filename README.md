## üß≠ Overview

**Conscious Information Geometry (CIG)** proposes that:

- **Informational integration** among agents forms the foundation of structure.  
- **Geometry** emerges from informational dependence via Fisher information metrics.  
- **Curvature** corresponds to changes in integration (Hessians of total correlation).  
- **Gravity** appears as *induced dynamics* from fluctuations on the informational manifold.  
- **Spacetime** is the *projection* of deeper informational processes.  

CIG offers a **neutral, epistemic, and classical** framework capable of connecting:

- General Relativity  
- Quantum Theory (via potential quantum information extensions)  
- Information Geometry  
- Cognitive and conscious processes  

This repository contains documentation, diagrams, code examples, and supporting material for the CIG preprint.

## üß© Core Idea

CIG is built on a **four-level ontology**:

Consciousness (phenomenal domain, not modeled directly)

Information (statistical structure, agents, correlations)

Geometry (emergent via Fisher metric & integration)
